@gyselroth/icon-collection v1.0.12

Overview

This library provides an icon collection based on svg.

Installation

npm --save install @gyselroth/icon-collection

Usage example

<div class="gr-icon gr-i-file"></div>

Colorize your icons

You can use an icon with any color you like. If you do you have to switch from the css class `gr-icon` to `gr-masgr-icon`. An icon with gr-masgr-icon takes the color from `background-color`.

<div style="background-color: #F01;" class="gr-masgr-icon gr-i-file"></div>

Browser support

If you want to use color masks, please see browser support for css masks
